# multica-health-manager

React + Tailwind 组件化原型，覆盖登录、用户端、兽医端、CMS、运营总览页面。

## 技术栈
- React 18 + TypeScript + Vite
- Tailwind CSS
- React Router

## 本地运行
```bash
npm install
npm run dev
```

## 生产构建
```bash
npm run build
npm run preview
```

## 页面路由
- `/login` 登录页
- `/dashboard` 运营总览
- `/user` 用户端
- `/vet` 兽医端
- `/cms` CMS 管理端
